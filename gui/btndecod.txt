Screen/Button "TOM MIX DECODER" sheet for Trima GUI

The button push designation occupies a four byte integer
The first two bytes (leftmost) contain the hexadecimal number of the screen
The last two bytes (rightmost) contain the hexadecimal number of the button

 Example. 0014002c, screen = 14 = GUI_SCREEN_ALARM
                    button = 2c = GUI_BUTTON_ALARM_CONTINUE
