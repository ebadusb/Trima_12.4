1. Open the psd file in Photoshop
2. Make sure only the desired layers are visible
NOTE: Do not save the psd file after this step!
3. Image | Mode | Indexed Color
  3a. Ok to flatten layers and discard hidden layers
  3b. Palette = Exact; Forced = Black and White; Transparency = no; OK
4. File | Save As...
  4a. BMP file type
  4b. Windows format; 8-bit Depth; OK
